# Philosophy Works Ranking Survey

An interactive web application that allows users to rank philosophical works and discover their preferred schools of philosophy through an engaging survey experience.

## Features

- **30 Philosophical Works**: Curated selection spanning 5 major schools of philosophy
  - Ancient Greek Philosophy
  - Rationalism
  - Empiricism
  - Existentialism
  - Pragmatism

- **Interactive Rating System**:
  - Rate 5 works at a time with upvote/downvote buttons
  - Visual feedback for selections
  - Progress tracking through all 30 works

- **Data Visualization**:
  - Results displayed using Google Charts API
  - Bar graph showing upvotes by school of philosophy
  - Beautiful, responsive design
